How to dump the game files (xci/nsp) using Yuzu ("RomFS")

Preparations

You should already have the game as a .xci or .nsp and added to Yuzu. Make sure the game works in Yuzu.

Dumping

Right click the game in Yuzu and click Dump RomFS -> Dump RomFS. Yuzu will not respond for a couple of seconds and then ask what dump mode. Choose Full and press OK. Yuzu will then dump the game for you, which will take about 5-15 minutes.

When finished, Yuzu will prompt you that the operation was successfull and when you press ok, open the location of the dumped game in the explorer. The dumped game should be in the Yuzu folder under /dump/0100F2C0115B6000/romfs. You can open the Yuzu folder by pressing File -> Open yuzu Folder in Yuzu.

The full path of the dumped game should look something like this: C:/Users/USERNAME/AppData/Roaming/yuzu/dump/0100F2C0115B6000/romfs (replace USERNAME with your username).

Important Considerations

Many of the files are still compressed. The compressed files can be identified by their .zs file extension.
